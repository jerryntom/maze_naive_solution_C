```
#     #*******#*# #
#######*#######*# #
#b******#*******# #
#########*####### #
#   #*****#a****  #
# # #*#########*###
# # #*#*******#***#
### #*#*#####*###*#
#   #***#   #***#*#
# ####### #####*#*#
#              ***#
###################
```
a - entrance of the maze;
b - exit of the maze
